Introduction

What is Nailed

Nailed is a CSS framework made to speed up the development process. Unlike big frameworks, Nailed is designed for people who wants to go straight to the point. Nailed is minimalist and light as possible. The main goal is to offer developers a good working basis and some really useful functions without any frills. We are here to make it quick, and efficient.


You will find here all you need to know about Nailed, and if you need more informations or if you have feedbacks feel free to contact me.



Getting started

This framework is only an assistant. Nailed will manage some part of your work to enhance your productivity but won't replace your job. Take this as an exoskeleton but not as an autopilot. Let's code!



You can download Nailed by several ways. Choose the method that suits you the best.


URL LINK :

Rows

Setup

To add a row in your page, just put your content inside a <div> tag with a row class. Here is down below the best way to set it up.


<div class="row">
   <p>You nailed it!</p>
</div>


Content spacing

As you can see on the code down below, there is some col class that you don't know yet. No worries, we are going to learn them later on. Stick to the row properties for the moment.



START


The start class will align the elements on the left of the row. It is the default value. But if for any reason you have to force the left alignment, know that there is a property available for it.


<div class="row start">
   <div class="col col-25">.col</div>
   <div class="col col-25">.col</div>
   <div class="col col-25">.col</div>
</div>


END


The end class will align the elements on the right of the row.


<div class="row end">
   <div class="col col-25">.col</div>
   <div class="col col-25">.col</div>
   <div class="col col-25">.col</div>
</div>


SPREAD


The spread class will space out the elements all along the row.


<div class="row spread">
   <div class="col col-25">.col</div>
   <div class="col col-25">.col</div>
   <div class="col col-25">.col</div>
</div>


CENTER


The center class will gather the elements in the center of the row.


<div class="row center">
   <div class="col col-25">.col</div>
   <div class="col col-25">.col</div>
   <div class="col col-25">.col</div>
</div>

Columns

Setup

To add a column in your row, just put a <div> tag with a col class. By default, a col will cover all the space available in the container. If you have more than one col in, the space will shared between them into equal parts.


<div class="row">
   <div class="col">.col</div>
   <div class="col">.col</div>
</div>


Dimensions

COL-0~100


You can add col-0~100 class, which will define the col width in percentage between 0 and 100. Integer numbers supported only.


<div class="row spread">
   <div class="col col-50">.col-50</div>
   <div class="col col-20">.col-20</div>
</div>


Notice that Col-0~100 has a forced width. If you place a col-50 and a col-20 in your row, 30% are still available. You can leave it empty if you want some space, or add regular cols to fill it.
